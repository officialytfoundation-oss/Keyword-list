# 4digit.txt (Mobile PIN Database)

• Format: 4-digit numeric codes

• Range: 0000 to 9999

• Total Entries: 10,000 combinations

• Common Uses:

  • Mobile phone PINs/Screen locks
  
  • Basic numeric password systems
  
  • Simple authentication systems
  
  • ATM PIN combinations

 # 6digit.txt (Extended Password Database)

• Format: 6-digit numeric codes

• Range: 000000 to 999999

• Total Entries: 1,000,000 combinations

• Common Uses:

  • Advanced numeric passwords
  
  • Some mobile device patterns
  
  • Banking OTP simulations
  
  • System authentication codes
